# Example Data

## sample.fasta
Small protein database (Ubiquitin, HBB, Myoglobin) for testing the search pipeline.
Source: UniProtKB/Swiss-Prot. These are real protein sequences from public databases.

## mzML files
No example mzML is included due to file size.
You can obtain test mzML files from:
- PRIDE Archive: https://www.ebi.ac.uk/pride/archive
- MassIVE: https://massive.ucsd.edu

For top-down proteomics examples, search for datasets tagged with "top-down" or "intact protein".

## Running with Demo Mode
If you don't have real mzML data, use the Demo engine in the UI.
It will generate synthetic results for workflow testing — clearly labeled as DEMO.
